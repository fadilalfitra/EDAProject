## Latar Belakang
Pada mulanya repositories ini saya buat sebagai repository pertama yang saya kerjakan secara serius di Github.
Saya bermaksud menjadikan repository ini sebagai tempat menyimpan dan mencari code, serta menjadi tempat saya menyimpan project-project apa saja yang sudah saya kerjakan meskipun tidak sampai ke tahapan pemodelan. Karena seringkali project yang cuma dijalankan sampai tahapan EDA menjadi project yang dianggap remeh, padahal EDA sangatlah penting dan menjadi tonggak dasar suatu analisis sebelum dilakukan ke tahapan selanjutnya. Bahkan, jikalau suatu dataset tidak sampai pada tahap pemodelan, EDA sendiri sudah memberikan informasi tambahan yang cukup untuk dijadikan dasar dalam mengambil keputusan.

## Apa itu EDA
Exploratory Data Analysis (EDA) adalah suatu analisis yang dilakukan pada tahapan-tahapan awal melakukan analisis. Analisis ini biasanya dilakukan sebelum memasuki tahap modeling, karena dilakukan untuk mendapatkan informasi atau gambaran umum mengenai dataset yang sedang dikerjakan. Untuk machine learning ataupun model regresi, tahapan ini biasanya dilakukan untuk mengetahui struktur tiap variabel/feature sekaligus mencari tahu hubungan/korelasi antar variabel.
Oleh karena itu, EDA biasanya sangat membutuhkan statistik dasar dan grafik-grafik berbasis analisis deskriptif, misalnya boxplot, barplot, line chart, leaf and steam, matriks korelasi, heatmap, dan sebagainya.

## Bagaimana repository ini disusun
Repository ini berisi folder-folder yang memuat bahan-bahan masing-masing project yang dilakukan dengan metode EDA. Setiap project akan memiliki Readme nya sendiri sebagai bahan bacaan yang akan mempermudah user lainnya memahami project tersebut. Selain itu, saya akan mengusahakan agar setiap project juga memiliki setiap hal yang dibutuhkan untuk mereplikasi riset tersebut, misalnya dengan menyediakan data analisis, ataupun metadatanya.
